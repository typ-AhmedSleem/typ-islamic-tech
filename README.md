### 2. · [islamic-toolkit-kt](https://github.com/typ-AhmedSleem/islamic-toolkit-kt)

> A pure **Kotlin** library providing essential Islamic utilities for Android and JVM projects — Hijri calendar conversion, prayer time calculations, Qibla direction, and more.

- **Tech Stack:** Kotlin
- **Target:** Android · JVM
- **License:** ✅ Open Source

---

### 3. · [islamic-toolkit-kmp](https://github.com/typ-AhmedSleem/islamic-toolkit-kmp)

> The **Kotlin Multiplatform** edition of Islamic Toolkit — same powerful utilities, extended to run across Android, iOS, and all KMP targets from a single shared module.

- **Tech Stack:** Kotlin Multiplatform (KMP)
- **Target:** Android · iOS · KMP
- **License:** ✅ Open Source

- **License:** ✅ Open Source
